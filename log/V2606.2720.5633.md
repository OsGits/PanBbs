# V2606.2720.5633 更新日志

## 📋 概述

本次更新围绕 **数据缓存管理** 和 **版本管理系统** 两大主题进行了重构与增强，新增后台缓存设置面板、支持覆盖模式的数据同步、以及统一的版本检测机制。

---

## ✨ 新增功能

### 1. 后台缓存设置面板

在系统设置页（`?a=settings`）新增 **缓存设置** 标签页，管理员可自由配置：

- **缓存网盘类型**：支持半角逗号分隔多个类型（如 `115,guangya,quark`），系统将仅从远程 API 缓存指定类型的资源到本地 `data/oss.json`
- **每种类型最大缓存数**：数字输入框，限制每类网盘在本地 JSON 中的最大记录条数

> 📁 涉及文件：`admin/page_settings.php`、`admin/actions.php`、`data/data.php`

### 2. 覆盖模式数据同步

`api.php` 中的 `fetchAndStore()` 函数新增第 5 个参数 `$overwrite`（默认 `false`）：

| 模式 | 行为 | 使用场景 |
|------|------|----------|
| 增量模式 (`false`) | 新数据与旧数据合并去重，新数据排在前面 | 后台手动抓取（`action=fetch_data`） |
| **覆盖模式** (`true`) | 丢弃旧数据，仅保留本次抓取结果 | `ting.php` 定时触发 |

`ting.php` 作为定时任务入口，已切换为**覆盖模式**，每次激活将完全替换本地 JSON 缓存数据，确保数据整洁无冗余。

> 📁 涉及文件：`api.php`、`ting.php`

### 3. 统一版本管理系统

将版本获取逻辑收敛到单一文件 `version.php`：

- **本地版本**：定义常量 `PANBBS_LOCAL_VERSION`（当前版本号）
- **远程版本**：通过 GitHub Releases API（`OsGits/PanBbs`）获取最新版本标签
- **1 小时缓存**：远程版本数据缓存到 `data/version_cache.txt`，避免频繁请求 GitHub API
- **HTTP 兜底**：`file_get_contents` 获取失败时自动降级为 cURL

所有页面（前台 `index.php`、后台 `admin.php`、模板 `_head.php`）统一从此文件获取版本号。

### 4. 版本查询 API

前台新增 `/?a=version` 端点，返回 JSON 格式的版本信息：

```json
{
  "code": 0,
  "data": {
    "local": "V2606.2720.5633",
    "remote": "v1.0.0"
  }
}
```

### 5. 前端版本更新提示

用户打开版权弹窗时，前端会懒加载版本信息，自动比较本地版本与远程版本，不一致时显示 **"有新版本可用"** 提示。

> 📁 涉及文件：`template/app.js`

### 6. 后台版本更新页

后台新增版本更新页面（`?a=version`）：
- 显示当前版本号
- 支持直接修改本地版本号
- 展示 `log/` 目录下的更新日志列表（按文件名倒序，最多 10 条）

> 📁 涉及文件：`admin/page_version.php`、`admin/actions.php`

## 🐛 修复

- **修复常量重复定义警告**：`template/_head.php` 使用 `require` 加载 `version.php` 时，若 `index.php` 已加载则触发 `PANBBS_LOCAL_VERSION already defined` 错误。改为 `defined()` 守卫判断后解决。

## 📐 架构说明

### 数据流向

```
远程 API → api.php(解析/展平) → 按类型分类 → URL去重 → oss.json(本地缓存)
                                                          ↓
                                    index.php/?a=api → 前端卡片展示
                                    index.php/?a=search → 远程实时搜索(不缓存)
```

### 前后端版本链路

```
GitHub Releases API → version.php(1h缓存) → PANBBS_REMOTE_VERSION 常量
                                              ↓
                              index.php/?a=version → 前端 fetchLatestVersion()
                                                       ↓
                                                 版权弹窗展示 + 更新提示
```
