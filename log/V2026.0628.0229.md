### 2026.0628.0205 更新日记

#### 版本弹窗 UI 重构

##### 问题
右下角「关于」弹窗中，本地版本和远程版本的展示方式为横向并排卡片，视觉效果不够精致。

##### 改动

**前端 - `template/_modal_version.php`**
- 布局从横向并排改为纵向堆叠的卡片式设计
- 每个版本卡片新增图标（本地：用户图标 / 远程：地球图标）和标题头
- 中间用连接线 + 圆形状态指示器替代原来的箭头分隔符
- 状态文字优化：「与云端版本不一致，建议同步更新！」→「检测到新版本，建议更新！」

**样式 - `template/style.css`**
- 卡片样式：带边框卡片 + hover 微浮动效果
- 图标带彩色圆角背景（本地紫色 / 远程橙色）
- 连接器状态节点支持三种状态：默认灰色（获取中）、橙色脉冲（有新版本）、绿色（已是最新）
- 版本不一致提示增加浅色背景底板
- 整体尺寸缩小：卡片 max-width 280→220、padding 缩小、字号 22→18、图标 26→22、连接器 26→20

**JS - `template/app.js`**
- 新增 `connectorStatus` 元素引用
- `showLatestVersion()` 根据版本对比结果自动切换连接器状态类名（`is-outdated` / `is-updated`）
- `showVersionFail()` 清除连接器状态

#### 首页列表 - 详情弹窗

##### 问题
首页卡片「打开链接」按钮直接外链跳转，用户无法在站内预览资源详情。

##### 改动

**新增 - `template/_modal_detail.php`**
- 资源详情弹窗模板，无边框毛玻璃风格
- 包含：关闭按钮、类型图标、标题、类型标签、正文内容、标签区、操作区（复制源链接 + 密码复制 + 添加时间）

**前端 - `template/app.js`**
- 卡片「打开链接」按钮改为「查看详情」按钮，触发详情弹窗
- 每张卡片添加 `data-idx` 属性关联 `allItems` 数组索引
- 新增 `openDetailModal(item)` 函数：动态渲染图标、标题、类型、正文、标签、链接、密码、时间
- 新增 `closeDetailModal()` 函数
- 新增 `bindDetailTrigger()` 批量绑定详情按钮事件
- 弹窗中「打开链接」改为「复制源链接」，点击复制到剪切板
- 复制源链接提示：「引擎提示：链接已复制！」
- `fallbackCopy()` 新增自定义提示参数，区分源链接复制和密码复制

**样式 - `template/style.css`**
- 新增 `.detail-overlay` / `.detail-modal` / `.detail-header` / `.detail-body` / `.detail-meta` / `.detail-actions` 全套样式
- `.link-btn` 兼容 `<button>` 标签（新增 `border: none`、`cursor: pointer`、`font-family: inherit`）
- `.detail-link-btn` 同样兼容 `<button>` 标签

**入口 - `template/home.php`**
- 引入 `_modal_detail.php`

#### Toast 提示位置调整

- Toast 从页面顶部 (`top: 20px`) 移到底部 (`bottom: 24px`)
- 动画方向从向上滑入改为向下滑入
