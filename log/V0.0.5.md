# V0.0.5 更新记录

## 1. 搜索弹窗 - 网盘类型筛选优化

### 筛选逻辑调整
- 复选框**默认全部不勾选**（不勾选 = 搜索全部网盘）
- 勾选部分网盘 = 仅搜索勾选的网盘
- 新增"全选/取消全选"按钮，一键切换所有复选框状态

### 按钮尺寸缩小
- 网盘选择标签字号从 `12px` → `11px`
- 内边距从 `5px 10px` → `2px 8px`
- 标签间距从 `6px` → `4px`
- 全选按钮同步缩小，与标签风格统一

## 2. 搜索结果按远程API时间排序

- `flattenRecord()` 函数不再使用 `date('Y-m-d H:i:s')` 当前时间
- 改为从远程API数据中提取每条记录的 `datetime` 字段作为 `add_time`
- 搜索结果按 `add_time` 倒序排列（从新到旧）

## 3. 后端搜索默认行为调整

- 搜索接口未指定 `types` 参数时，默认搜索**全部 13 种网盘类型**（之前默认只搜 3 种）

## 涉及文件

| 文件 | 改动说明 |
|------|----------|
| `template/_modal_search.php` | 复选框默认不勾选，恢复全选/取消全选按钮 |
| `template/style.css` | 缩小网盘选择标签样式，恢复全选按钮样式 |
| `template/app.js` | 恢复全选/取消全选相关逻辑和事件绑定 |
| `api.php` | `flattenRecord()` 使用远程API的 `datetime` 字段 |
| `index.php` | 搜索未指定类型时使用全部 13 种网盘 |
